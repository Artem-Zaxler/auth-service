Мини-сервис сервис авторизации
Описание:
Разработать мини-сервис, выполняющий роль авторизационного шлюза, который
позволяет производить управление пользователями, выдавать отчетность и вход в другие
сервисы.
Стек:
- Основа: Symfony + SQL (MSSQL, PostgreSQL)
- Дополнительно: Docker, Git
Роли:
- ADMIN (все возможные действия)
- USER (авторизация для других сервисов)
Сущности:
 - Пользователи
- Сессии
- Токены
Возможные действия:
 - Создание пользователя
- Просмотр и редактирования информации о пользователе
- Блокировка пользователя
- Отчеты
- Списки
Требования:
- Стек: PHP 8.2+, Symfony 6.4/7.x
- Архитектура: тонкие контроллер, разделение уровней.
- Транзакции и конкурентный доступ
- Логирование.
- Доктрина и БД: миграции, индексы.
- Обработка ошибок: единый JSON-формат ошибок, корректные HTTP-коды,
валидация входных DTO.
- Формы доступа: ЛК, Rest API, Auth2
- Документация
- Git репозиторий с пошаговым сохранением по мере реализации функционала.
Желательно:
- Docker,
- Тесты.
